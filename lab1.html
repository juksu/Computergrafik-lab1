Size:
Width of playing field (in Blocks): 10
(Changing this value will reset the playing field)

Controls:
  • "->" or "r": move the object drawn one unit to the right
  • "<-" or "l": move the object drawn one unit to the left
  • "/\" or "u": move the object drawn one unit up
  • "\/" or "d": move the object drawn one unit down
  • "1": rotate the object drawn 90 degrees counterclockwise
  • "3": rotate the object drawn 90 degrees clockwise
  • "enter": spawn a new tetromino
